# open-cart-extensions
A collection of extensions for OpenCart

OneClick Installer. Replaces the multi-step plugin installer for OpenCart. Once it is installed using the usual method, all installations/updates are simple drag-drop.

Admin Contrast. An extension that changes the contrast index of the admin section, making it easier to read for lower vision users.

Image Grid. An extension that reformats the layout and drag-drop functions on product images

Option Image Grid. Same as Image Grid, but for Options.

Option Image Swap. Implements drag-drop re-ordering of option images.

Sticky Admin Header. Let's face it: the OpenCart interface is the best that the early 2000s had to offer. This extension fixes the admin header in place so thataccess to those functions isn't hidden when scrolling down.
